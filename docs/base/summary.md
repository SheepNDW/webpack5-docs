# 總結

本章節我們學會了 Webpack 基本使用，掌握了以下功能：

1. 兩種開發模式

- 開發模式：程式碼能編譯自動化運行
- 生產模式：程式碼編譯優化輸出

2. Webpack 基本功能

- 開發模式：可以編譯 ES Module 語法
- 生產模式：可以編譯 ES Module 語法，壓縮 js 程式碼

3. Webpack 配置文件

- 5 個核心概念
  - entry
  - output
  - loader
  - plugins
  - mode
- devServer 配置

4. Webpack 腳本指令用法

- `webpack` 直接打包輸出
- `webpack serve` 啟動DevServer，内存編譯打包沒有輸出
